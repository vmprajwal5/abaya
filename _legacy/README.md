# Abaya Clothing E-Commerce Website

## Project Description
"Abaya Clothing" is a premium e-commerce web application designed for selling modest fashion wear. The project aims to provide a seamless, elegant shopping experience with a focus on usability and aesthetic appeal, inspired by modern minimalist fashion sites.

## Features
- **Responsive Design:** Fully functional on Mobile, Tablet, and Desktop.
- **Product Catalog:** Dynamic grid layout for displaying clothing.
- **Shopping Cart:** JavaScript-based cart interaction.
- **Admin Panel:** Static dashboard structure for managing products.
- **Elegant UI:** Custom CSS implementation with a Gold/Navy color scheme.

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Icons:** FontAwesome
- **Fonts:** Google Fonts (Playfair Display, Lato)
- **Editor:** VS Code

## How to Run Locally
1. Download or Clone the repository.
2. Open the folder `abaya-clothing-project` in VS Code.
3. Install the "Live Server" extension in VS Code.
4. Right-click on `index.html` and select "Open with Live Server".
